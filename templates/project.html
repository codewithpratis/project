Project Overview

Project Overview

    We started out with the jupyter notebook to import the CSV file that we downloaded from Kaggle DataSet. We used pandas, matplotlib, and Numpy methods to manupulate the data. After manipulating the data, we had alot of questions that we wanted to answer. Some of the questions that we wanted to tackle were:
  • How many bookings were canceled and not?
  • Which hotel is more popular?
  • How much of bookings were done each year?
  • Which year was the busiest?
  • Did Deposit Type have any effects on cancelations?
  • What is a correlations between lead time and cancelations?
  • How did people booked the room? What's the most popular way to book the room?
  • Out of singles, couples, and family, who were the most popular ones to stay in the hotel?
    • However, the main questions is how can these all be the predictor of the cancelations that happened over the years?
    In order to answer this question, we used Machine Learning method by applying Machine Learning Libraries such as:
  • Sklearn
  • Tensor Flow
    • So we used these ML libraries to predict the cancelations.

    Factors Influencing Hotel Cancelations
     Observation Analysis
  • We found out that the prediction was 77.4% accurate.
  • We used pandas get dummies function, which made it easier since it encodes the categorical data.
  • We decided to limit the number of epochs to 20 after looking at the original 100.
  • The accuracy and loss changed slightly after epoch 6.
  • When we plot the data, 77% of the data seems to be accurately predicted.

    • Obstacles/Issues Encountered
  • We had little hard time using the get dummies function to convert the categorical data into the number of feature to be used.
  • Each value became its own column.
  • Doing it this way, could have made for a huge number of columns depending on the number of different values of categorical data in each column.
  • We also had a difficulty finding the meaning of calculated result loss of 0.45, which we calculated using categorical cross entropy
